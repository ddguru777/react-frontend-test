# React FrontEnd Test Project

Project Was Built In:
 * ReactJS
 * Redux
 * Webpack 4 
 * Babel 7
 * React Material UI
 * Bootstrap 4 
 * SCSS, CSS Support
 * Code Formatter (Prettier)
 * Eslint configured extended with Airbnb style guide & support for prettier
 
## Installation
### Install with NPM
```
npm install
npm start
```
### Install with YARN
```
yarn install
yarn start
```
